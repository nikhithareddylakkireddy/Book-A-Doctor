# Doctor Appointment Booking System

## Live Demo
https://book-a-doctor.streamlit.app/

## Features
🏥 Professional Hospital Management Interface
👤 Patient Registration and Secure Login
🔐 Password Protection with Authentication
📅 Online Doctor Appointment Booking
🆔 Automatic Appointment ID Generation
👨‍⚕ Department-wise Doctor Selection
📧 Email Confirmation for Booked Appointments
📋 View Appointment Details
❌ Cancel Appointments
📊 Interactive Admin Dashboard
📈 Appointment Statistics (Total, Pending, Completed, Cancelled)
📑 View All Patient Appointment Records
📥 Download Appointment Records as CSV
🔍 Search Appointments by Registered Email
🏥 Multiple Medical Departments
💾 SQLite Database Integration
🎨 Clean, Responsive, and User-Friendly Interface
☁️ Cloud Deployment using Streamlit Community Cloud
🔄 Real-Time Appointment Status Management
👩‍💻 Developer-Friendly Project Structure
